DSP EXECUTABLE LOADER
[LNK_040_DES.pdf]
This document describes the overall design and architecture of the Loader used to parse and load DSP binaries. It lists the interfaces exposed by the loader and also describes the overall design for implementation of these interfaces.
It would be the starting point for users trying to implement their own parser and/ or loader.
OS SPECIFIC DOCUMENTS
OS Adaptation Layer for Linux
[LNK_024_DES.pdf]
This document describes the overall design and architecture of the OS Adaptation Layer for Linux.
HARDWARE SPECIFIC DOCUMENTS
Shared Memory IOM Driver Design for OMAP5910/5912
[LNK_019_DES.pdf]
This document describes the design of the link driver for data communication between the GPP and DSP. This link driver utilizes the shared memory architecture of the OMAPTM devices.
Since the design is reasonably simple, this document is a good starting point for users trying to implement a different link driver.
PORTING DOCUMENTS
PORTING GUIDE
[LNK_017_PRT.pdf]
Provides recommendations and guidelines for the developers to port DSP/BIOSTM LINK to either a different GPP OS, a different platform or a different physical link.
